# St. Andrew's Academy — Alumni Portal

**School:** St. Andrew's Academy, Candaba, Pampanga, Philippines  
**Established:** 1946  
**Portal Version:** 1.0 — May 2026

## Hosting Instructions (GitHub Pages)

1. Upload `index.html` to your GitHub repository
2. Go to **Settings → Pages**
3. Set Source: `main` branch, `/ (root)` folder
4. Your portal will be live at: `https://[username].github.io/[repo-name]/`

## Files
- `index.html` — Complete self-contained portal (all data, images, styles, scripts embedded)
- `README.md` — This file

## Features
- 📊 Overview Dashboard with Summary & Statistics
- 📜 History Dashboard with photo gallery
- 🎓 Graduation Class Dashboard (Batch 1986 — Sections A–G)
- 👥 Alumni Database (Add/Edit/Delete/Export/Backup)
- 📸 Memory Lane (Events with photo albums)
- ⚙️ Settings with backup/restore

## Important Notes
- Data edits are saved to **browser localStorage** only
- Use the **Export/Backup** buttons to save data permanently
- Re-upload `index.html` after major data updates to preserve data across devices
